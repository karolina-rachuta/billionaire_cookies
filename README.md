# Billionaire Cookies Website
Business website for cookies company.

## Tech stack
- React js
- Contentful
- SCSS
- RWD


## How to run:
- `git clone git@github.com:karolina-rachuta/billionaire_cookies.git`
- `npm install`
- `npm start`
